1. SÍMBOLOS TERMINAIS (Tokens -> palavras):

    AUTOMACAO     ->  automacao
    FIM           ->  fim
    QUANDO        ->  quando
    SE            ->  se
    SENAO         ->  senao
    ENTAO         ->  entao
    LIGAR         ->  ligar
    DESLIGAR      ->  desligar
    ESPERAR       ->  esperar
    NOTIFICAR     ->  notificar
    DEFINIR       ->  definir
    CHAMAR        ->  chamar
    MUDAR         ->  mudar
    PARA          ->  para
    HORARIO       ->  horario
    OU            ->  ou
    E             ->  e
    MODO          ->  modo

    ENTIDADE_ID   ->  identificador de entidade (ex: sensor.temperatura_sala, light.luz_teto)
    TEMPO         ->  valor numérico com unidade de tempo (ex: 10s, 5min, 1h, 500ms)
    NUMERO        ->  valor numérico inteiro ou decimal (ex: 25, 3.14)
    STRING        ->  texto entre aspas duplas (ex: "on", "Porta aberta")

    MAIOR         ->  >
    MENOR         ->  <
    IGUAL         ->  ==
    MAIOR_IGUAL   ->  >=
    MENOR_IGUAL   ->  <=
    DIFERENTE     ->  !=

    PONTO_VIRGULA ->  ;

    COMENTARIO    ->  # até o fim da linha
    Espaços, tabs, quebras de linha

2. SÍMBOLOS NÃO-TERMINAIS (Regras):

    programa,
    lista_automacoes,
    automacao,
    gatilho_bloco,
    lista_gatilhos,
    gatilho,
    condicao_opt,
    lista_condicoes,
    condicao,
    operador,
    expressao,
    lista_acoes,
    acao,
    modo_opt,
    bloco_se,
    bloco_senao_opt

3. SÍMBOLO INICIAL:

    programa

4. REGRAS DE PRODUÇÃO:

    R1:  programa         -> lista_automacoes

    R2:  lista_automacoes -> lista_automacoes PONTO_VIRGULA automacao
    R3:  lista_automacoes -> automacao

    R4:  automacao        -> AUTOMACAO STRING gatilho_bloco condicao_opt ENTAO lista_acoes modo_opt FIM

    R5:  gatilho_bloco    -> QUANDO lista_gatilhos

    R6:  lista_gatilhos   -> lista_gatilhos OU gatilho
    R7:  lista_gatilhos   -> gatilho

    R8:  gatilho          -> ENTIDADE_ID MUDAR PARA expressao
    R9:  gatilho          -> HORARIO STRING                      # ex.: 'horario "05:00:00"'

    R10: condicao_opt     -> SE lista_condicoes
    R11: condicao_opt     -> e  (vazio)

    R12: lista_condicoes  -> lista_condicoes E condicao           # múltiplas condições (AND)
    R13: lista_condicoes  -> condicao

    R14: condicao         -> ENTIDADE_ID operador expressao

    R15: operador         -> MAIOR
    R16: operador         -> MENOR
    R17: operador         -> IGUAL
    R18: operador         -> MAIOR_IGUAL
    R19: operador         -> MENOR_IGUAL
    R20: operador         -> DIFERENTE

    R21: expressao        -> NUMERO
    R22: expressao        -> STRING

    R23: lista_acoes      -> lista_acoes acao
    R24: lista_acoes      -> acao

    R25: acao             -> LIGAR ENTIDADE_ID
    R26: acao             -> DESLIGAR ENTIDADE_ID
    R27: acao             -> ESPERAR TEMPO
    R28: acao             -> NOTIFICAR STRING PARA STRING    # ex.: 'notificar "Luzes ajustadas" para "mobile_zfold4"'
    R29: acao             -> DEFINIR ENTIDADE_ID expressao
    R30: acao             -> CHAMAR ENTIDADE_ID ENTIDADE_ID expressao  # ex.: 'chamar media_player.volume_set media_player.music_frame 0.09'
    R31: acao             -> CHAMAR ENTIDADE_ID ENTIDADE_ID            # ex.: 'chamar automation.trigger automation.cozinha_ligar_luzes'
    R32: acao             -> bloco_se

    R33: bloco_se         -> SE lista_condicoes ENTAO lista_acoes bloco_senao_opt FIM

    R34: bloco_senao_opt  -> SENAO lista_acoes
    R35: bloco_senao_opt  -> e  (vazio)

    R36: modo_opt         -> MODO STRING                     # ex.: 'modo "reiniciar"' ou 'modo "fila"'
    R37: modo_opt         -> e  (vazio)                      # padrão: single
